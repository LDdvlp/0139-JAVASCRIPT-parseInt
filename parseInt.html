The parseInt function (method) only attempts to extract the first value of a string.


    const stringedAge1 = '50.05 is age';
    const stringedAge2 = '50,05 is age';
    const stringedAge3 = '50/05 is age';
    const stringedAge4 = '50 100 150 200';
    const stringedAge5 = '50istheage';
    console.log(parseInt(stringedAge1));
    console.log(parseInt(stringedAge2));
    console.log(parseInt(stringedAge3));
    console.log(parseInt(stringedAge4));
    console.log(parseInt(stringedAge5));

Paragraph #06 Code :

Base de numération de 2 à 36.
Attention ! La base par défaut n'est pas 10.
Ce paramètre doit toujours être utilisé, en effet s'il n'est pas spécifié,
le comportement de la fonction n'est pas garanti et peut varier d'une plate-forme à une autre.
Si l'argument string commence avec "0x" ou "0X", la base considérée sera la base 16 (hexadécimale).
Si l'argument string commence avec "0", la base considérée sera la base 8 (octale) ou la base 10 (décimale).
Si l'argument string commence avec une autre valeur, la base considérée sera la base 10.
